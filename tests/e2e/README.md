# E2E (Playwright)

## Requisitos

Defina credenciais de teste com acesso ao sistema:

```bash
export E2E_USER_EMAIL="seu-email@exemplo.com"
export E2E_USER_PASSWORD="sua-senha"
```

Opcional:

```bash
export E2E_PORT=4173
export E2E_BASE_URL="http://127.0.0.1:4173"
```

## Comandos

```bash
npm run e2e:install
npm run e2e
```

Modo UI:

```bash
npm run e2e:ui
```
